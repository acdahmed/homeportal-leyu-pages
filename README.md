# homeportal-leyu-pages

本项目是一个用于归档和发布多个独立 HTML 页面的仓库。页面内容涵盖各类静态展示、工具页面或信息聚合页，不针对任何特定域名或具体网站。

## 项目简介

该仓库的主要目的是集中管理一批独立、可访问的 HTML 页面，方便通过 GitHub Pages 等服务进行静态托管。每个页面均为自包含的 HTML 文件，可直接在浏览器中打开使用。

## 目录说明

```
.
├── index.html          # 项目首页或导航页
├── pages/              # 存放各个独立 HTML 页面
│   ├── page1.html
│   ├── page2.html
│   └── ...
├── assets/             # 静态资源（图片、样式、脚本等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

- `pages/`：核心目录，每个文件为一个独立页面。
- `assets/`：页面引用的公共资源文件，按类型归类存放。

## 页面归档说明

所有页面均为静态 HTML，不依赖后端服务。页面内容可能包括信息展示、功能工具或实验性页面，供日常参考或分发使用。每个页面在提交时均经过基础可用性检查。

## 维护说明

- 新增页面请放置在 `pages/` 目录下，并确保资源引用路径正确。
- 页面更新时请同步修改相关资源文件，并保持目录结构清晰。
- 不建议在本仓库中存放敏感信息或动态内容。
- 欢迎提交 Issue 或 Pull Request 以改进页面内容或结构。

## 许可

本项目仅作为页面归档用途，不附带特定许可协议。各页面内容可能引用第三方资源，请遵守相应版权规定。
